문제
회원가입시 각 필드마다 유효성 검사를 하여 처리할 수 있도록 만들어라.
1. ID는 영어소문자/숫자만 가능해야함(4~12자리이며 소문자로 시작)
2. PW는 영어 대/소문자, 숫자만 입력가능해야함(8~12자리)
3. PW와 PW_re는 같아야 함.
4. Name은 무조건 한글만 가능해야 함.(2글자 이상)
5. 주소에는 영어 사용 불가(공백 불가)
6. 우편번호는 무조건 숫자만 입력 가능해야함
7. 이메일은 @가 포함되어야 함 (@앞에가 영문자,숫자로 4~12)
8. 전화번호는 첫번째필드 010, 두번째필드 4자리 세번쨰 4자리 ( 단 모두 숫자만 가능)
