자바스크립트 객체

객체 선언 및 호출

객체는 키 값을 사용하여 value을 식별함. value에는 모든 자료형이 올 수 있다

객체의 키는 모든 문자열을 사용할 수 있다
키로 사용할 수 없는 단어를 키로 ㅅ가용한 경우에는 무조건 대괄호를 사용해야 객체의 요소에 접근할 수 있다.

객체의 메소드

객체의 value중 함수 자료형인 value를 메소드라 함.

객체의 in과 with 키워드

in: 객체 내부에 해당 속성이 있는지 확인하는 키워드
with: 호출시 코드를 줄여주는 키워드 (객체명 생략가능)





객체의 키,값 추가와 제거

처음 객체 생성 이후 키,값을 추가하거나 제거하는 것을 동적으로 할 수 있음.
