윈도우 객체

window 객체는 자바스크립트의 최상위 객체이며 BOM, DOM으로 나뉜다.

Window 객체

브라우저 창에 대한 설정을 하는 객체






















navigator 객체

웹 페이지를 실행하고 있는 브라우저에 대한 정보를 가지고 있는 객체

screen 객체

웹 브라우저 화면이 아닌 운영체제 화면의 속성을 가진 객체

★location 객체

브라우저 주소 표시줄과 관련된 객체(페이지 이동)

history 객체

현재 창에서 사용자의 방문 기록을 저장함. 뒤로가기, 앞으로가기, 새로고침 가능
